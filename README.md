# 🧠 EEG-Based State Classification using Machine Learning

## Overview
This project focuses on classifying human states using EEG signal features and machine learning models. It includes preprocessing, exploratory data analysis, model training, and performance evaluation using separate train and test datasets.

## Problem
EEG signals contain meaningful patterns that can be used for classification tasks. The goal of this project is to build a machine learning pipeline that predicts the target state from EEG-based features.

## Approach
- Loaded separate train and test datasets
- Explored and cleaned the data
- Encoded labels for classification
- Trained and compared multiple machine learning models
- Evaluated performance using accuracy, classification report, and confusion matrix

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Results
The models were compared based on test accuracy and classification metrics. The project demonstrates a complete EEG classification workflow from preprocessing to evaluation.
